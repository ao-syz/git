# 🌳 分支学习计划 — 软件开发技能树

> "编程不是一条直线，而是一棵树。选择你的分支，但别忘了主干。"

---

## 主干：编程基础（必修）

这是所有分支的根基，必须先扎实。

### 第一阶段：语法与逻辑（1-2 周）

- [ ] 学习一门编程语言的基础语法（推荐 Python 或 JavaScript）
- [ ] 理解变量、数据类型、运算符
- [ ] 掌握条件判断（if/else）和循环（for/while）
- [ ] 理解函数的定义和调用

### 第二阶段：数据结构（2-3 周）

- [ ] 数组/列表的操作（增删改查）
- [ ] 字典/对象的结构化存储
- [ ] 字符串处理和正则表达式入门
- [ ] 基本算法：排序、查找

### 第三阶段：面向对象与设计（2-3 周）

- [ ] 理解类、对象、继承、多态
- [ ] 理解封装和抽象
- [ ] 学习设计模式（单例、工厂、观察者）
- [ ] 代码重构的基本原则

---

## 分支 A：前端开发 🎨

> 负责用户看得见、摸得着的界面

### A1. HTML/CSS 基础（2 周）

- [ ] HTML5 语义化标签
- [ ] CSS 选择器、盒模型、布局（Flexbox / Grid）
- [ ] 响应式设计和媒体查询
- [ ] 常见 UI 组件实现（导航栏、卡片、表单）

### A2. JavaScript 深入（3-4 周）

- [ ] DOM 操作和事件处理
- [ ] 异步编程（Promise / async await）
- [ ] ES6+ 新特性（解构、箭头函数、模块化）
- [ ] 浏览器存储（LocalStorage / SessionStorage / Cookie）

### A3. 前端框架（3-4 周）

- [ ] Vue.js 或 React 选择一个深入学习
- [ ] 组件化开发和状态管理
- [ ] 路由管理（Vue Router / React Router）
- [ ] 构建工具（Vite / Webpack）

### A4. 进阶前端（持续学习）

- [ ] TypeScript 类型系统
- [ ] 前端性能优化
- [ ] 单元测试（Jest / Vitest）
- [ ] 服务端渲染（Nuxt / Next.js）

---

## 分支 B：后端开发 ⚙️

> 负责数据处理、业务逻辑和系统架构

### B1. 服务器基础（2 周）

- [ ] HTTP 协议深入理解
- [ ] RESTful API 设计规范
- [ ] JSON 数据格式和接口调试（Postman / curl）
- [ ] 环境变量和配置管理

### B2. 后端框架（3-4 周）

选择一个阵营：

**Node.js 路线：**
- [ ] Express / Koa / Nest.js
- [ ] 中间件和路由

**Python 路线：**
- [ ] Django / FastAPI / Flask
- [ ] ORM 和数据库迁移

**Java 路线：**
- [ ] Spring Boot
- [ ] Maven / Gradle 构建

### B3. 数据库（2-3 周）

- [ ] SQL 基础（CRUD、JOIN、索引）
- [ ] 关系型数据库（MySQL / PostgreSQL）
- [ ] NoSQL 数据库（MongoDB / Redis）
- [ ] 数据库设计范式

### B4. 进阶后端（持续学习）

- [ ] 身份认证（JWT / OAuth / Session）
- [ ] 消息队列（RabbitMQ / Kafka）
- [ ] 缓存策略（Redis 缓存、CDN）
- [ ] 微服务架构入门

---

## 分支 C：DevOps & 工具链 🔧

> 提升开发效率和代码质量

### C1. 版本控制（1 周）

- [ ] Git 工作流（分支策略、Rebase vs Merge）
- [ ] GitHub / GitLab 协作流程
- [ ] Pull Request 和 Code Review

### C2. 开发与部署（2-3 周）

- [ ] Linux 基础命令
- [ ] Docker 容器化
- [ ] CI/CD 流水线（GitHub Actions / Jenkins）
- [ ] 云服务部署（Vercel / AWS / 阿里云）

### C3. 代码质量（持续）

- [ ] 代码规范（ESLint / Prettier / Black）
- [ ] 单元测试和集成测试
- [ ] 日志管理和监控

---

## 分支 D：数据与 AI 🤖

> 数据驱动的世界

### D1. 数据分析（2-3 周）

- [ ] Pandas / NumPy 数据处理
- [ ] 数据可视化（Matplotlib / ECharts）
- [ ] 描述性统计基础

### D2. 机器学习入门（4-6 周）

- [ ] 监督学习 vs 无监督学习
- [ ] Scikit-learn 常用算法
- [ ] 模型评估指标
- [ ] 简单的深度学习（PyTorch / TensorFlow）

---

## 📅 推荐学习路线

### 初学者的 6 个月路线

| 月份 | 重点 |
|------|------|
| 第 1 月 | 主干：编程基础 |
| 第 2 月 | 主干：数据结构 + 选择 A/B 分支 |
| 第 3 月 | 深入选定的分支（前端或后端） |
| 第 4 月 | 学习数据库 + 完成一个实战项目 |
| 第 5 月 | 学习 DevOps 工具链（C 分支） |
| 第 6 月 | 综合项目：把前端 + 后端 + 部署串起来 |

---

## 🎯 学习原则

1. **不要同时走太多分支** — 先精通一个，再扩展
2. **项目驱动学习** — 每学一个知识点，写一个小项目
3. **读优秀的代码** — GitHub 上的开源项目是最好的老师
4. **记录与复盘** — 写学习笔记，定期回顾
5. **社区参与** — 遇到问题先搜索，再提问，最后贡献
